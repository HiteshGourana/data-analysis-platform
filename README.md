# 📊 Automated Data Analysis Platform

A **Streamlit-based data analysis platform** that allows users to upload datasets, perform **Exploratory Data Analysis (EDA)**, clean data, and visualize insights interactively.

---

## 🚀 Features

* 📂 Upload dataset (CSV)
* 📊 Exploratory Data Analysis (EDA)
* 🧹 Data Cleaning Tools

  * Remove duplicates
  * Handle missing values
  * Change data types
  * Drop columns
* 📈 Data visualization
* ⚡ Interactive interface using Streamlit

---

## 🛠️ Tech Stack

* **Python**
* **Pandas**
* **NumPy**
* **Matplotlib**
* **Streamlit**

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/your-username/automated-data-analysis-platform.git
```

2. Move into the project directory

```
cd automated-data-analysis-platform
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Run the Streamlit app

```
streamlit run app.py
```

---

## 📊 Usage

1. Upload a dataset (CSV file)
2. Perform **EDA** to understand the dataset
3. Use the **data cleaning page** to fix missing values, duplicates, and datatypes
4. Visualize insights from the dataset
